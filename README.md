## Data

`uuid`

`positionId`            职位ID

`positionName`          职位名称

`positionType`          职位类型

`positionAdvantage`     职位诱惑

------

`companyName`           公司名称

`companyShortName`      公司简称

`companySize`           公司规模

`companyLabelList`      公司标签列表

`industryField`         行业领域

`financeStage`          融资阶段

------

`city`                  城市

`district`              区域

`businessZones`         商业区

`address`               具体地址

------

`salary`                薪水

`workYear`              工作经验

`education`             学历要求

`jobNature`             工作性质

`jobDescription`        职位描述

------

`createTime`            创建时间

`jobUrl`                招聘URL

`collectionTime`        采集时间
